# Unity Open MCP

Unity Open MCP предоставляет ИИ-агентам типизированный, защищённый шлюзами
набор инструментов для работы с проектами Unity.

MCP-сервер открывает **250+ инструментов** для типизированных рабочих процессов
редактора, шлюзов и валидации, анализа ресурсов, диагностики и встроенных
доменных групп.

## Ключевые возможности

- Безопасные изменения с автоматической валидацией, чекпоинтами, дельтами,
  регрессионными проверками и точечными исправлениями.
- Структурированный поиск, инспекция, пересериализация и анализ ссылок на
  ресурсы.
- Живой мост Unity, пакетный откат для поддерживаемых инструментов и офлайн-ридеры.
- Типизированные рабочие процессы редактора и встроенных доменов, доступные через
  группируемые по сессии наборы инструментов.
- Unity Hub Pro для управляемой установки и сопровождающих рабочих процессов.

Полный каталог и контракты — в [docs/api/mcp-tools.md](docs/api/mcp-tools.md)
(на английском).

## Быстрая установка

Требуется **Unity 2022.3 LTS или новее**.

1. **Простейший способ (ИИ-агент):** вставьте этот промпт в ваш ИИ-клиент
   (Cursor, Claude, …) и позвольте ему установить всё за вас:

```text
Установите Unity Open MCP в этот проект Unity, точно следуя
https://raw.githubusercontent.com/AlexeyPerov/Unity-Open-MCP/master/docs/setup/agent-setup.md
Выполняйте каждый шаг агента самостоятельно; останавливайтесь и сообщайте мне,
только когда требуется действие пользователя (USER ACTION).
Если этот монорепозиторий уже открыт локально, прочитайте docs/setup/agent-setup.md
с диска, а не загружайте по сети.
```

Полная процедура: [Установка агентом](docs/ru/setup/agent-setup.md).

2. **Вручную:** отредактируйте пакеты и конфигурацию MCP-клиента самостоятельно с
   помощью [Ручной установки](docs/ru/setup/manual-setup.md).
3. **Локальный чекаут:** соберите и запустите репозиторий с помощью
   [Установки для разработки](docs/ru/setup/development-setup.md).
4. **Unity Hub Pro:** используйте UI-поток из
   [Установки через мастер](docs/ru/setup/wizard-setup.md).

## Unity Hub Pro

Unity Hub Pro — это настольное приложение-компаньон для Unity Open MCP. Оно помогает
управлять проектами, запускать мастер AI Setup и вести сопровождающие рабочие процессы
из единого интерфейса.
